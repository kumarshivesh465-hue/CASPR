# CASPR - Climate Analysis of Soil and Prediction Rover

## Overview

CASPR (Climate Analysis of Soil and Prediction Rover) is a smart agriculture platform that combines IoT, cloud computing, satellite imagery, and web technologies to monitor soil health and environmental conditions in real time.

The system uses an ESP32-based rover equipped with multiple sensors to collect soil and climate data. This information is uploaded to Firebase Realtime Database, fused with satellite-derived insights, and displayed through a real-time web dashboard to support data-driven agricultural decision-making.

---

## Features

* Real-time soil monitoring
* Soil moisture measurement
* Temperature and humidity tracking
* NPK sensor integration
* pH monitoring
* Firebase Realtime Database integration
* Cloud-based data synchronization
* Satellite imagery analysis
* Data fusion between rover and satellite observations
* Precision agriculture recommendations
* Real-time web dashboard

---

## System Architecture

ESP32 Rover → Firebase Realtime Database → Cloud Functions → Web Dashboard

### Data Flow

1. Rover collects environmental and soil data.
2. Sensor readings are uploaded to Firebase.
3. Cloud Functions process and fuse rover and satellite data.
4. Recommendations are generated automatically.
5. Dashboard visualizes all information in real time.
